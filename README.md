# Turtle Racing Game 🐢

A simple turtle racing game built with Python's built-in Turtle graphics library. Players place a bet on which turtle will win the race and watch the race unfold in real time.

## Features

- Four racing turtles with different colors
- User betting system
- Randomized turtle movement
- Race winner detection
- Simple graphical interface using Turtle Graphics

## Technologies Used

- Python 3
- Turtle Graphics
- Random Module

## How It Works

1. The program asks the user to place a bet on a turtle color.
2. Four turtles line up at the starting position.
3. Each turtle moves forward by a random distance every iteration.
4. The first turtle to cross the finish line wins.
5. The winner is compared with the user's bet.
